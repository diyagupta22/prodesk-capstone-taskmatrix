🚀 TaskMatrix – Project Management Tool

📌 Project Description
TaskMatrix is a full-stack web-based project management application designed to help teams efficiently manage tasks, track progress, and collaborate in real time. It provides a Kanban-style interface similar to Jira and Asana, enabling users to organize tasks into different stages such as "To Do", "In Progress", and "Done".



❗ Problem Statement
Managing tasks in teams becomes difficult without a structured system. Teams often struggle with:
- Lack of task visibility
- Poor collaboration
- Missed deadlines
- Unorganized workflows

TaskMatrix solves these problems by providing a centralized platform for task tracking and team collaboration.



🎯 Target Users
- Software development teams
- Students working on group projects
- Freelancers managing multiple tasks
- Small teams and startups

🚀 Core Features

🔐 Authentication
- User registration and login (JWT-based authentication)
- Secure password handling

📋 Task Management
- Create, edit, and delete tasks
- Assign tasks to users
- Set priority (High, Medium, Low)

📊 Kanban Board
- Drag and drop tasks between:
  - To Do
  - In Progress
  - Done

👥 Team Collaboration
- Add team members to projects
- Assign roles (Admin / Member)

🔔 Activity Feed
- Track recent actions like:
  - Task created
  - Task updated
  - Task moved
